Features

AI Resume Optimization

Analyzes resume against any job description using Gemini 2.0 and GPT-4, suggesting 8-20 targeted improvements per application with ATS score feedback.

Application Tracker

Full-text PostgreSQL search across 360+ job applications with real-time status updates and automated Google Drive document management.

Recruiter Automation

AI-powered LinkedIn search + Gmail API for personalized outreach email generation with automated draft creation and 4-score candidate analysis.

Dual-Server Architecture

Separate Express.js servers for optimization and tracking, allowing independent scaling and zero-downtime deployments for each service.

Google Workspace Integration

Automated sync with Google Drive, Docs, and Gmail APIs for seamless document management across 360+ applications.

Vertex AI Scoring

Google Cloud Vertex AI powers advanced candidate scoring with a 4-dimension fit score across skills, experience, education, and culture.
